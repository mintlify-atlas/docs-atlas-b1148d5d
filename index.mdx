KolkataRail Intelligence helps commuters on Kolkata suburban trains make safer, smarter travel decisions. The platform fuses real-time crowd reports, historical data, trend signals, and weather to predict crowding levels — and provides a conversational AI system for filing FIR complaints in English, Hindi, or Bengali.

## What KolkataRail Intelligence does

The platform runs as three coordinated FastAPI services:

- **Portal** (port 5000) — serves the web UI and proxies all API traffic to the two backend services
- **Crowd Intelligence** (port 8000) — train search, crowd prediction, live reporting, and ML insights
- **NirbhayaSathi** (port 8001) — conversational FIR filing, AI categorization, and a police dashboard

## Get started

<Steps>
  <Step title="Install dependencies">
    Create a Python virtual environment and install the required packages.
  </Step>
  <Step title="Configure environment variables">
    Set Neo4j credentials and optional API keys for weather and Groq AI.
  </Step>
  <Step title="Start all three services">
    Launch NirbhayaSathi on port 8001, Crowd Intelligence on 8000, and the portal on 5000.
  </Step>
  <Step title="Open the portal">
    Navigate to `http://localhost:5000/portal.html` to access the passenger-facing interface.
  </Step>
</Steps>
